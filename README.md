# SG Income Tax Estimator (YA 2026)

A lightweight, client-side tax estimator for Singapore tax residents filing for Year of Assessment 2026. Built as a single HTML file with no dependencies, no server, and no data collection.

## Features

- Covers employment, rental, trade/business, and other taxable income
- Models all major personal reliefs — CPF, SRS, NSman, child, parent, spouse, life insurance, and more
- Handles the $80,000 personal relief cap and Parenthood Tax Rebate (PTR) separately
- CPF relief auto-calculated from salary inputs, with manual override
- Live tax estimate visible at all times via sticky preview bar
- Fully client-side — nothing is sent anywhere

## Usage

No installation needed. Just open `index.html` in any modern browser.

If serving via GitHub Pages, the file will be available at:
`https://economist1895.github.io/iit-estimator/`

## Disclaimer

This tool is for estimation purposes only. It does not constitute tax advice. Always verify your reliefs and income figures against official IRAS guidance before filing.

## Data & Privacy

All calculations run entirely in your browser. No data is collected, stored, or transmitted.
